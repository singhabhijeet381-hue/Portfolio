💼 Personal Portfolio Website

This is a **Flask-based Personal Portfolio Website** designed to showcase my projects, skills, and experience as a developer. It provides a clean and responsive interface for visitors to learn more about me and explore my work.

---

## 🚀 Features

- 🌐 Dynamic portfolio built with Flask
- 📄 Multiple pages (Home, About, Projects, Contact)
- 🎨 Clean and responsive UI using HTML & CSS
- 📁 Organized structure with templates and static files
- ⚡ Lightweight and fast performance
